How to use {{title}}

  1. Upload a CCW-R Excel file containing the quote to be extracted.
  2. An Excel file with prefix "quosal_" will automatically be downloaded to your computer.
    • The Excel file will have three tabs:
      1. Invoice Line Item Detail - lists Service Ordering SKUs and quantities.
      2. Quote Line Item Detail - lists all the details of the Smartnet quote including Serial nos., Contract, Start, and End date.
      3. CCW-R # - lists the raw data from the original quote for reference.

Creating a Smartnet renewal quote in Quosal

  1. Click "New Quote" and choose the "SMARTnet Renewals" quote template.
    • On Prepare Content tab, there will be two Products tabs:
      1. Invoice Line Item Detail - paste the contents of the Excel sheet with the same name here.
      2. Quote Line Item Detail - paste the contents of the Excel sheet with the same name here.
  2. To properly display the columns in the quote, choose the appropriate mapping by clicking "Load Mapping" and choosing:
    • Smartnet Invoice Line Mapping for the Invoice Line Item Detail tab.
    • Smartnet Quote Line Mapping for the Quote Line Item Detail tab.
  3. Publish and Deliver.
