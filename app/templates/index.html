How to use {{title}}

  1. Upload the soitems.csv file exported from the opportunity's Product tab on ConnectWise.
  2. An Excel file named "proserv_converted.xlsx" will be automatically downloaded to your computer.
    • The Excel file will have one sheet with two tables:
      1. First Table - The first table lists the raw data from the original export file with some lines highlighted in red.
      2. Second Table - The second table will list the PKG-PRO-PROFSVCS SKUs adjusted to reflect 50-20-20-10 progress billing.
